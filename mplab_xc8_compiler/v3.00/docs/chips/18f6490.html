18F6490 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Oscillator : External RC oscillator, port function on RA6
// Fail-Safe Clock Monitor Enable : Fail-Safe Clock Monitor disabled
// Internal External Switch Over Mode : Oscillator Switchover mode disabled
#pragma config OSC = RCIO, FCMEN = OFF, IESO = OFF
#pragma config <setting>=<literal constant>
For example:
// Oscillator : External RC oscillator, port function on RA6
// Fail-Safe Clock Monitor Enable : Fail-Safe Clock Monitor disabled
// Internal External Switch Over Mode : Oscillator Switchover mode disabled
#pragma config OSC = 0x7, FCMEN = 0x0, IESO = 0x0
#pragma config <register>=<literal constant>
For example:
// Oscillator : External RC oscillator, port function on RA6
// Fail-Safe Clock Monitor Enable : Fail-Safe Clock Monitor disabled
// Internal External Switch Over Mode : Oscillator Switchover mode disabled
#pragma config CONFIG1H = 0x7
For example:
// IDLOC @ 0x200000
#pragma config IDLOC0 = 0x255

EC EC oscillator, CLKO function on RA6
RC External RC oscillator, CLKO function on RA6
HS HS oscillator
XT XT oscillator
LP LP oscillator
FCMEN Fail-Safe Clock Monitor Enable
OFF Fail-Safe Clock Monitor disabled
ON Fail-Safe Clock Monitor enabled
IESO Internal External Switch Over Mode
OFF Oscillator Switchover mode disabled
ON Oscillator Switchover mode enabled

PWRT Power Up Timer
OFF PWRT disabled
ON PWRT enabled
BOREN Brown Out Detect
SBORDIS Brown-out Reset enabled in hardware only (SBOREN is disabled)

3 VBOR set to 2.1V
