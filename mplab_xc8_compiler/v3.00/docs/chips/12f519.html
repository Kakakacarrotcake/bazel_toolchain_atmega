12F519 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Oscillator Selection bits : EXTRC With 1 ms DRT
// Watchdog Timer Enable bit : Enabled
// Code Protection bit : Code protection off
// Master Clear Enable bit : RB3/MCLR Functions as MCLR
// Internal Oscillator Frequency Select bit : 8 MHz INTOSC Speed
// Code Protection bit - Flash Data Memory : Code protection off
#pragma config FOSC = EXTRC, WDTE = ON, CP = OFF, MCLRE = ON, IOSCFS = 8MHz, CPDF = OFF
#pragma config <setting>=<literal constant>
For example:
// Oscillator Selection bits : EXTRC With 1 ms DRT
// Watchdog Timer Enable bit : Enabled
// Code Protection bit : Code protection off
// Master Clear Enable bit : RB3/MCLR Functions as MCLR
// Internal Oscillator Frequency Select bit : 8 MHz INTOSC Speed
// Code Protection bit - Flash Data Memory : Code protection off
#pragma config FOSC = 0x3, WDTE = 0x1, CP = 0x1, MCLRE = 0x1, IOSCFS = 0x1, CPDF = 0x1
#pragma config <register>=<literal constant>
For example:
// Oscillator Selection bits : EXTRC With 1 ms DRT
// Watchdog Timer Enable bit : Enabled
// Code Protection bit : Code protection off
// Master Clear Enable bit : RB3/MCLR Functions as MCLR
// Internal Oscillator Frequency Select bit : 8 MHz INTOSC Speed
// Code Protection bit - Flash Data Memory : Code protection off
#pragma config CONFIG = 0xFFF
For example:
// IDLOC @ 0x440
#pragma config IDLOC0 = 0x4095

Configuration Registers & Settings

Register: CONFIG @ 0xFFF
FOSC Oscillator Selection bits
LP LP Osc With 18 ms DRT
XT XT Osc With 18 ms DRT
INTRC INTRC With 1 ms DRT
EXTRC EXTRC With 1 ms DRT
WDTE Watchdog Timer Enable bit
ON Enabled
OFF Disabled
CP Code Protection bit
OFF Code protection off
ON Code protection on
MCLRE Master Clear Enable bit
ON RB3/MCLR Functions as MCLR
OFF RB3/MCLR Functions as RB3
IOSCFS Internal Oscillator Frequency Select bit
8MHz 8 MHz INTOSC Speed
4MHz 4 MHz INTOSC Speed
CPDF Code Protection bit - Flash Data Memory
OFF Code protection off
ON Code protection on
Register: IDLOC0 @ 0x440
Register: IDLOC1 @ 0x441
Register: IDLOC2 @ 0x442
Register: IDLOC3 @ 0x443
