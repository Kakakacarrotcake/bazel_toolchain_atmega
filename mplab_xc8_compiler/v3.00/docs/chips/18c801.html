18C801 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Power-up Timer Enable bit : PWRT disabled
// External Bus Data Width bit : 16-bit External Bus mode
#pragma config PWRT = OFF, BW = 16
#pragma config <setting>=<literal constant>
For example:
// Power-up Timer Enable bit : PWRT disabled
// External Bus Data Width bit : 16-bit External Bus mode
#pragma config PWRT = 0x1, BW = 0x1
#pragma config <register>=<literal constant>
For example:
// Power-up Timer Enable bit : PWRT disabled
// External Bus Data Width bit : 16-bit External Bus mode
#pragma config CONFIG2L = 0x41

PIC-AS config Usage

config "<setting>"="<named value>"
For example:
; Power-up Timer Enable bit : PWRT disabled
; External Bus Data Width bit : 16-bit External Bus mode
config "PWRT" = "OFF", "BW" = "16"
config "<setting>"=<literal constant>
For example:
; Power-up Timer Enable bit : PWRT disabled
; External Bus Data Width bit : 16-bit External Bus mode
config "PWRT" = 0x1, "BW" = 0x1
config "<register>"=<literal constant>
For example:
; Power-up Timer Enable bit : PWRT disabled
; External Bus Data Width bit : 16-bit External Bus mode
config "CONFIG2L" = 0x41

Configuration Registers & Settings

Register: CONFIG1H @ 0x300001
OSC Oscillator Selection bits
RC RC Oscillator
HS HS Oscillator
EC EC Oscillator
LP LP Oscillator
Register: CONFIG2L @ 0x300002
PWRT Power-up Timer Enable bit
OFF PWRT disabled
ON PWRT enabled
BW External Bus Data Width bit
16 16-bit External Bus mode
8 8-bit External Bus mode
Register: CONFIG2H @ 0x300003
WDT Watchdog Timer Enable bit
ON WDT enabled
OFF WDT disabled (control is placed on the SWDTEN bit)
WDTPS Watchdog Timer Postscale Select bits
128 1:128
64 1:64
32 1:32
16 1:16
8 1:8
4 1:4
2 1:2
1 1:1
Register: CONFIG4L @ 0x300006
STVR Stack Full/Underflow RESET Enable bit
ON Stack Full/Underflow will cause RESET
OFF Stack Full/Underflow will not cause RESET
