ATXMEGA64A1 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Watchdog Timeout Period : 8 cycles (8ms @ 3.3V)
// Watchdog Window Timeout Period : 8 cycles (8ms @ 3.3V)
#pragma config WDPER = WDPER_8CLK, WDWPER = WDWPER_8CLK
#pragma config <setting>=<literal constant>
For example:
// Watchdog Timeout Period : 8 cycles (8ms @ 3.3V)
// Watchdog Window Timeout Period : 8 cycles (8ms @ 3.3V)
#pragma config WDPER = 0x0, WDWPER = 0x0
