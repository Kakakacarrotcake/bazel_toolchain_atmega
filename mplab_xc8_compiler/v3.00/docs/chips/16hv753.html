16HV753 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// FOSC: Oscillator Selection bit : EC oscillator mode. CLKIN function on RA5/CLKIN
// Watchdog Timer Enable bit : Watchdog Timer enabled
// Power-up Timer Enable bit : Power-up Timer disabled
// MCLR/VPP Pin Function Select bit : MCLR pin is MCLR function with internal weak pullup
// Code Protection bit : Program memory code protection is disabled
// Brown-out Reset Enable bits : BOR enabled
// Flash Program Memory Self Write Enable bit : Flash self-write protection off
// Clock Out Enable bit : CLKOUT function disabled. CLKOUT pin acts as I/O
#pragma config FOSC0 = EC, WDTE = ON, PWRTE = OFF, MCLRE = ON, CP = OFF, BOREN = EN, WRT = OFF, CLKOUTEN = OFF
#pragma config <setting>=<literal constant>
For example:
// FOSC: Oscillator Selection bit : EC oscillator mode. CLKIN function on RA5/CLKIN

// Code Protection bit : Program memory code protection is disabled
// Brown-out Reset Enable bits : BOR enabled
// Flash Program Memory Self Write Enable bit : Flash self-write protection off
// Clock Out Enable bit : CLKOUT function disabled. CLKOUT pin acts as I/O
#pragma config FOSC0 = 0x1, WDTE = 0x1, PWRTE = 0x1, MCLRE = 0x1, CP = 0x1, BOREN = 0x3, WRT = 0x3, CLKOUTEN = 0x1

For example:
// FOSC: Oscillator Selection bit : EC oscillator mode. CLKIN function on RA5/CLKIN

Configuration Registers & Settings

Register: CONFIG @ 0x2007
FOSC0 FOSC: Oscillator Selection bit
EC EC oscillator mode. CLKIN function on RA5/CLKIN
INT Internal oscillator mode. I/O function on RA5/CLKIN
WDTE Watchdog Timer Enable bit
ON Watchdog Timer enabled
OFF Watchdog Timer disabled
PWRTE Power-up Timer Enable bit
OFF Power-up Timer disabled
ON Power-up Timer enabled
MCLRE MCLR/VPP Pin Function Select bit
ON MCLR pin is MCLR function with internal weak pullup
OFF MCLR pin is alternate function
CP Code Protection bit
OFF Program memory code protection is disabled
ON Program memory code protection is enabled
BOREN Brown-out Reset Enable bits
EN BOR enabled
SLEEP_DIS BOR enabled during operation and disabled in Sleep
DIS BOR disabled
WRT Flash Program Memory Self Write Enable bit
OFF Flash self-write protection off
FOURTH 000h to 0FFh self-write protected
HALF 000h to 1FFh self-write protected
ALL 000h to 3FFh self-write protected
CLKOUTEN Clock Out Enable bit
OFF CLKOUT function disabled. CLKOUT pin acts as I/O
ON CLKOUT function enabled. CLKOUT pin is CLKOUT
Register: IDLOC0 @ 0x2000
Register: IDLOC1 @ 0x2001
Register: IDLOC2 @ 0x2002
Register: IDLOC3 @ 0x2003
