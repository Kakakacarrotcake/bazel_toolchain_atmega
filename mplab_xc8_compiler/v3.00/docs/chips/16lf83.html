16LF83 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Oscillator Selection bits : RC oscillator
// Watchdog Timer : WDT enabled
// Power-up Timer Enable bit : Power-up Timer is disabled
// Code Protection bit : Code protection disabled
#pragma config FOSC = EXTRC, WDTE = ON, PWRTE = OFF, CP = OFF
#pragma config <setting>=<literal constant>
For example:
// Oscillator Selection bits : RC oscillator
// Watchdog Timer : WDT enabled
// Power-up Timer Enable bit : Power-up Timer is disabled
// Code Protection bit : Code protection disabled
#pragma config FOSC = 0x3, WDTE = 0x1, PWRTE = 0x1, CP = 0x3FF
#pragma config <register>=<literal constant>
For example:
// Oscillator Selection bits : RC oscillator
// Watchdog Timer : WDT enabled
// Power-up Timer Enable bit : Power-up Timer is disabled
// Code Protection bit : Code protection disabled
#pragma config CONFIG = 0x3FFF
For example:
// IDLOC @ 0x2000
#pragma config IDLOC0 = 0x127

Configuration Registers & Settings

FOSC Oscillator Selection bits
EXTRC RC oscillator
HS HS oscillator
XT XT oscillator
LP LP oscillator
WDTE Watchdog Timer
ON WDT enabled
OFF WDT disabled
PWRTE Power-up Timer Enable bit
OFF Power-up Timer is disabled
ON Power-up Timer is enabled
CP Code Protection bit
OFF Code protection disabled
ON All program memory is code protected
