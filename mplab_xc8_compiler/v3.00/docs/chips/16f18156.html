16F18156 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// External Oscillator Selection bits : EC (external clock) above 16 MHz
// Reset Oscillator Selection bits : EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit : CLKOUT function is disabled; i/o or oscillator function on OSC2
// Clock Switch Enable bit : Writing to NOSC and NDIV is allowed
// VDD Range Analog Calibration Selection bit : Internal analog systems are calibrated for operation between VDD = 2.3 - 5.5V
// Fail-Safe Clock Monitor Enable bit : Fail-Safe Clock Monitor enabled
#pragma config FEXTOSC = ECH, RSTOSC = EXTOSC, CLKOUTEN = OFF, CSWEN = ON, VDDAR = HI, FCMEN = ON
#pragma config <setting>=<literal constant>
For example:
// External Oscillator Selection bits : EC (external clock) above 16 MHz
// Reset Oscillator Selection bits : EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit : CLKOUT function is disabled; i/o or oscillator function on OSC2
// Clock Switch Enable bit : Writing to NOSC and NDIV is allowed
// VDD Range Analog Calibration Selection bit : Internal analog systems are calibrated for operation between VDD = 2.3 - 5.5V
// Fail-Safe Clock Monitor Enable bit : Fail-Safe Clock Monitor enabled
#pragma config FEXTOSC = 0x7, RSTOSC = 0x7, CLKOUTEN = 0x1, CSWEN = 0x1, VDDAR = 0x1, FCMEN = 0x1
#pragma config <register>=<literal constant>
For example:
// External Oscillator Selection bits : EC (external clock) above 16 MHz
// Reset Oscillator Selection bits : EXTOSC operating per FEXTOSC bits
// Clock Out Enable bit : CLKOUT function is disabled; i/o or oscillator function on OSC2
// Clock Switch Enable bit : Writing to NOSC and NDIV is allowed
// VDD Range Analog Calibration Selection bit : Internal analog systems are calibrated for operation between VDD = 2.3 - 5.5V
// Fail-Safe Clock Monitor Enable bit : Fail-Safe Clock Monitor enabled
#pragma config CONFIG1 = 0x3FFF
For example:
// IDLOC @ 0x8000
#pragma config IDLOC0 = 0x16383

Configuration Registers & Settings

Register: CONFIG1 @ 0x8007
FEXTOSC External Oscillator Selection bits
ECH EC (external clock) above 16 MHz
ECL EC (external clock) below 16 MHz
OFF Oscillator not enabled
HS HS (crystal oscillator) above 4MHz
XT XT (crystal oscillator) above 100kHz, below 4MHz
LP LP (crystal oscillator) optimized for 32.768kHz
RSTOSC Reset Oscillator Selection bits
EXTOSC EXTOSC operating per FEXTOSC bits
HFINTOSC_1MHz HFINTOSC (1MHz)
LFINTOSC LFINTOSC
SOSC SOSC
EXTOSC_4PLL EXTOSC with 4x PLL, with EXTOSC operating per FEXTOSC bits
HFINTOSC_16MHz HFINTOSC (16 MHz)
HFINTOSC_32MHz HFINTOSC (32 MHz)
CLKOUTEN Clock Out Enable bit
ON CLKOUT function is enabled; FOSC/4 clock appears at OSC2
OFF CLKOUT function is disabled; i/o or oscillator function on OSC2
CSWEN Clock Switch Enable bit
ON Writing to NOSC and NDIV is allowed
OFF The NOSC and NDIV bits cannot be changed by user software
VDDAR VDD Range Analog Calibration Selection bit
HI Internal analog systems are calibrated for operation between VDD = 2.3 - 5.5V
LO Internal analog systems are calibrated for operation between VDD = 1.8 - 3.6V
FCMEN Fail-Safe Clock Monitor Enable bit
ON Fail-Safe Clock Monitor enabled
OFF Fail-Safe Clock Monitor disabled
Register: CONFIG2 @ 0x8008
MCLRE Master Clear Enable bit
EXTMCLR If LVP = 0, MCLR pin is MCLR; If LVP = 1, RA3 pin function is MCLR
INTMCLR If LVP = 0, MCLR is port-defined function; If LVP = 1, RA3 pin function is MCLR
PWRTS Power-up Timer Selection bits
PWRT_OFF PWRT is disabled
PWRT_64 PWRT set at 64 ms
PWRT_16 PWRT set at 16 ms
PWRT_1 PWRT set at 1 ms
LPBOREN Low-Power BOR Enable bit
OFF ULPBOR disabled
ON ULPBOR enabled
BOREN Brown-out Reset Enable bits
ON Brown-out Reset enabled, SBOREN bit is ignored
NSLEEP Brown-out Reset enabled while running, disabled in sleep; SBOREN is ignored
SBOREN Brown-out reset enabled according to SBOREN bit
OFF Brown-out reset disabled
DACAUTOEN DAC Buffer Automatic Range Select Enable bit
OFF DAC Buffer reference range is determined by the REFRNG bit
ON DAC Buffer reference range is automatically determined by module hardware
BORV Brown-out Reset Voltage Selection bit
LO Brown-out Reset Voltage (VBOR) set to 1.9V
HI Brown-out Reset Voltage (VBOR) is set to 2.85V
ZCD ZCD Disable bit
OFF ZCD module is disabled; ZCD can be enabled by setting the ZCDSEN bit of ZCDCON
ON ZCD module is always enabled; ZCDMD and ZCDSEN bits are ignored
PPS1WAY PPSLOCKED One-Way Set Enable bit
ON The PPSLOCKED bit can be cleared and set only once after an unlocking sequence is executed; once PPSLOCKED is set, all future changes to PPS registers are prevented
OFF The PPSLOCKED bit can be set and cleared as needed (unlocking sequence is required)
STVREN Stack Overflow/Underflow Reset Enable bit
ON Stack Overflow or Underflow will cause a reset
OFF Stack Overflow or Underflow will not cause a reset
DEBUG Background Debugger
OFF Background Debugger disabled
ON background debugger enabled
