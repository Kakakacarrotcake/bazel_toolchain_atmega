10F204 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Oscillator : This is the only option. It is here for backward compatibility
// Watchdog Timer : WDT enabled
// Code Protect : Code protection off
// Master Clear Enable : GP3/MCLR pin function is MCLR
#pragma config OSC = IntRC, WDTE = ON, CP = OFF, MCLRE = ON
#pragma config <setting>=<literal constant>
For example:
// Oscillator : This is the only option. It is here for backward compatibility
// Watchdog Timer : WDT enabled
// Code Protect : Code protection off
// Master Clear Enable : GP3/MCLR pin function is MCLR
#pragma config OSC = 0x1, WDTE = 0x1, CP = 0x1, MCLRE = 0x1
#pragma config <register>=<literal constant>
For example:
// Oscillator : This is the only option. It is here for backward compatibility
// Watchdog Timer : WDT enabled
// Code Protect : Code protection off
// Master Clear Enable : GP3/MCLR pin function is MCLR
#pragma config CONFIG = 0xFFF
For example:
// IDLOC @ 0x100
#pragma config IDLOC0 = 0x4095

Configuration Registers & Settings

Register: CONFIG @ 0xFFF
OSC Oscillator
IntRC This is the only option. It is here for backward compatibility
WDTE Watchdog Timer
ON WDT enabled
OFF WDT disabled
CP Code Protect
OFF Code protection off
ON Code protection on
MCLRE Master Clear Enable
ON GP3/MCLR pin function is MCLR
OFF GP3/MCLR pin fuction is digital I/O, MCLR internally tied to VDD
Register: IDLOC0 @ 0x100
Register: IDLOC1 @ 0x101
Register: IDLOC2 @ 0x102
Register: IDLOC3 @ 0x103
