RF509AF Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Oscillator selection bits : external RC oscillator
// Watchdog timer enable bit : WDT enabled
// Code protection bit : Code protection off
// MCLR enable bit : MCLR pin enabled
#pragma config OSC = ExtRC, WDT = ON, CP = OFF, MCLRE = ON
#pragma config <setting>=<literal constant>
For example:
// Oscillator selection bits : external RC oscillator
// Watchdog timer enable bit : WDT enabled
// Code protection bit : Code protection off
// MCLR enable bit : MCLR pin enabled
#pragma config OSC = 0x3, WDT = 0x1, CP = 0x1, MCLRE = 0x1
#pragma config <register>=<literal constant>
For example:
// Oscillator selection bits : external RC oscillator
// Watchdog timer enable bit : WDT enabled
// Code protection bit : Code protection off
// MCLR enable bit : MCLR pin enabled
#pragma config CONFIG = 0xFFF
For example:
// IDLOC @ 0x400
#pragma config IDLOC0 = 0x4095

Configuration Registers & Settings

Register: CONFIG @ 0xFFF
OSC Oscillator selection bits
ExtRC external RC oscillator
LP LP oscillator
XT XT oscillator
IntRC internal RC oscillator
WDT Watchdog timer enable bit
ON WDT enabled
OFF WDT disabled
CP Code protection bit
OFF Code protection off
ON Code protection on
MCLRE MCLR enable bit
ON MCLR pin enabled
OFF MCLR tied to VDD, (Internally)
Register: IDLOC0 @ 0x400
Register: IDLOC1 @ 0x401
Register: IDLOC2 @ 0x402
Register: IDLOC3 @ 0x403
