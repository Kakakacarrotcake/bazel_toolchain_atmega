12F529T39A Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Oscillator : EXTRC With 1 ms DRT
// Watchdog Timer Enable : Enabled
// Configuration Word Parity : Parity bit set
// Master Clear Enable : RB3/MCLR Functions as MCLR
// Internal Oscillator Frequency Select : 8 MHz INTOSC Speed
// Code Protection bit - Flash Data Memory : Code protection off
// Code Protect State : Disabled
#pragma config OSC = EXTRC, WDT = ON, PARITY = SET, MCLRE = ON, IOSCFS = 8MHz, CPDF = OFF, CP = DISABLE
#pragma config <setting>=<literal constant>
For example:
// Oscillator : EXTRC With 1 ms DRT
// Watchdog Timer Enable : Enabled
// Configuration Word Parity : Parity bit set
// Master Clear Enable : RB3/MCLR Functions as MCLR
// Internal Oscillator Frequency Select : 8 MHz INTOSC Speed
// Code Protection bit - Flash Data Memory : Code protection off
// Code Protect State : Disabled
#pragma config OSC = 0x3, WDT = 0x1, PARITY = 0x1, MCLRE = 0x1, IOSCFS = 0x1, CPDF = 0x1, CP = 0xB
#pragma config <register>=<literal constant>
For example:
// Oscillator : EXTRC With 1 ms DRT
// Watchdog Timer Enable : Enabled
// Configuration Word Parity : Parity bit set
// Master Clear Enable : RB3/MCLR Functions as MCLR
// Internal Oscillator Frequency Select : 8 MHz INTOSC Speed
// Code Protection bit - Flash Data Memory : Code protection off
// Code Protect State : Disabled
#pragma config CONFIG = 0xDFF
For example:
// IDLOC @ 0x640
#pragma config IDLOC0 = 0x4095

Configuration Registers & Settings

Register: CONFIG @ 0xFFF
OSC Oscillator
LP LP Osc With 18 ms DRT
XT XT Osc With 18 ms DRT
INTRC INTRC With 1 ms DRT
EXTRC EXTRC With 1 ms DRT
WDT Watchdog Timer Enable
ON Enabled
OFF Disabled
PARITY Configuration Word Parity
SET Parity bit set
CLEAR Parity bit clear
MCLRE Master Clear Enable
ON RB3/MCLR Functions as MCLR
OFF RB3/MCLR Functions as RB3
IOSCFS Internal Oscillator Frequency Select
8MHz 8 MHz INTOSC Speed
4MHz 4 MHz INTOSC Speed
CPDF Code Protection bit - Flash Data Memory
OFF Code protection off
ON Code protection on
CP Code Protect State
DISABLE Disabled
ENABLE Enabled
