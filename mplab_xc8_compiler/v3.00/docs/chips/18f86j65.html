18F86J65 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Watchdog Timer Enable bit : WDT enabled
// Stack Overflow/Underflow Reset Enable bit : Reset on stack overflow/underflow enabled
// Extended Instruction Set Enable bit : Instruction set extension and Indexed Addressing mode enabled
// Background Debugger Enable bit : Background debugger disabled; RB6 and RB7 configured as general purpose I/O pins
#pragma config WDT = ON, STVR = ON, XINST = ON, DEBUG = OFF
#pragma config <setting>=<literal constant>
For example:
// Watchdog Timer Enable bit : WDT enabled
// Stack Overflow/Underflow Reset Enable bit : Reset on stack overflow/underflow enabled
// Extended Instruction Set Enable bit : Instruction set extension and Indexed Addressing mode enabled
// Background Debugger Enable bit : Background debugger disabled; RB6 and RB7 configured as general purpose I/O pins
#pragma config WDT = 0x1, STVR = 0x1, XINST = 0x1, DEBUG = 0x1
#pragma config <register>=<literal constant>
For example:
// Watchdog Timer Enable bit : WDT enabled
// Stack Overflow/Underflow Reset Enable bit : Reset on stack overflow/underflow enabled
// Extended Instruction Set Enable bit : Instruction set extension and Indexed Addressing mode enabled
// Background Debugger Enable bit : Background debugger disabled; RB6 and RB7 configured as general purpose I/O pins
#pragma config CONFIG1L = 0xE1

Configuration Registers & Settings

Register: CONFIG1L @ 0x17FF8
WDT Watchdog Timer Enable bit
ON WDT enabled
OFF WDT disabled (control is placed on SWDTEN bit)
STVR Stack Overflow/Underflow Reset Enable bit
ON Reset on stack overflow/underflow enabled
OFF Reset on stack overflow/underflow disabled
XINST Extended Instruction Set Enable bit
ON Instruction set extension and Indexed Addressing mode enabled
OFF Instruction set extension and Indexed Addressing mode disabled (Legacy mode)
DEBUG Background Debugger Enable bit
OFF Background debugger disabled; RB6 and RB7 configured as general purpose I/O pins
ON Background debugger enabled; RB6 and RB7 are dedicated to In-Circuit Debug
Register: CONFIG1H @ 0x17FF9
CP0 Code Protection bit
OFF Program memory is not code-protected
ON Program memory is code-protected
Register: CONFIG2L @ 0x17FFA
FOSC Oscillator Selection bits
ECPLL EC oscillator, PLL enabled and under software control, CLKO function on OSC2
EC EC oscillator, CLKO function on OSC2
HSPLL HS oscillator, PLL enabled and under software control
HS HS oscillator
FOSC2 Default/Reset System Clock Select bit
ON Clock selected by FOSC1:FOSC0 as system clock is enabled when OSCCON<1:0> = 00
OFF INTRC enabled as system clock when OSCCON<1:0> = 00
FCMEN Fail-Safe Clock Monitor Enable
ON Fail-Safe Clock Monitor enabled
OFF Fail-Safe Clock Monitor disabled
IESO Two-Speed Start-up (Internal/External Oscillator Switchover) Control bit
ON Two-Speed Start-up enabled
OFF Two-Speed Start-up disabled
Register: CONFIG2H @ 0x17FFB

1024 1:1024

Register: CONFIG3L @ 0x17FFC
Register: CONFIG3H @ 0x17FFD
CCP2MX ECCP2 MUX bit
ON ECCP2/P2A is multiplexed with RC1
OFF ECCP2/P2A is multiplexed with RE7 in Microcontroller mode (80-pin and 100-pin devices) or with RB3 in Extended Microcontroller mode (100-pin devices only)
ECCPMX ECCP MUX bit
ON ECCP1 outputs (P1B/P1C) are multiplexed with RE6 and RE5; ECCP3 outputs (P3B/P3C) are multiplexed with RE4 and RE3
OFF ECCP1 outputs (P1B/P1C) are multiplexed with RH7 and RH6; ECCP3 outputs (P3B/P3C) are multiplexed with RH5 and RH4
ETHLED Ethernet LED Enable bit
ON RA0/RA1 are multiplexed with LEDA/LEDB when Ethernet module is enabled and function as I/O when Ethernet is disabled
OFF RA0/RA1 function as I/O regardless of Ethernet module status
