18LF8585 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Oscillator : HS oscillator with SW enabled 4x PLL
// Osc. Switch Enable : Oscillator system clock switch option is disabled (main oscillator is source)
#pragma config OSC = HSSWPLL, OSCS = OFF
#pragma config <setting>=<literal constant>
For example:
// Oscillator : HS oscillator with SW enabled 4x PLL
// Osc. Switch Enable : Oscillator system clock switch option is disabled (main oscillator is source)
#pragma config OSC = 0xE, OSCS = 0x1
#pragma config <register>=<literal constant>
For example:
// Oscillator : HS oscillator with SW enabled 4x PLL
// Osc. Switch Enable : Oscillator system clock switch option is disabled (main oscillator is source)
#pragma config CONFIG1H = 0x2E
For example:
// IDLOC @ 0x200000
#pragma config IDLOC0 = 0x255

PIC-AS config Usage

config "<setting>"="<named value>"
For example:
; Oscillator : HS oscillator with SW enabled 4x PLL
; Osc. Switch Enable : Oscillator system clock switch option is disabled (main oscillator is source)
config "OSC" = "HSSWPLL", "OSCS" = "OFF"
config "<setting>"=<literal constant>
For example:
; Oscillator : HS oscillator with SW enabled 4x PLL
; Osc. Switch Enable : Oscillator system clock switch option is disabled (main oscillator is source)
config "OSC" = 0xE, "OSCS" = 0x1
config "<register>"=<literal constant>
For example:
; Oscillator : HS oscillator with SW enabled 4x PLL
; Osc. Switch Enable : Oscillator system clock switch option is disabled (main oscillator is source)
config "CONFIG1H" = 0x2E
For example:
; IDLOC @ 0x200000
config "IDLOC0" = 0x255

XT XT oscillator
LP LP oscillator

ON Timer1 oscillator system clock switch option is enabled (oscillator switching is enabled)
OFF Oscillator system clock switch option is disabled (main oscillator is source)

Register: CONFIG3H @ 0x300005
CCP2MX CCP2 Mux
ON CCP2 input/output is multiplexed with RC1
OFF CCP2 is multiplexed with RB3 in Extended Microcontroller, Microprocessor or Microprocessor with Boot Block mode. Or with RE7 in Microcontroller mode.
