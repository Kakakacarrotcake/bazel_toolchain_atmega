MCP19124 Support Information

XC8-CC #pragma config Usage

#pragma config <setting>=<named value>
For example:
// Watchdog Timer Enable bit : WDT enabled
// Power-up Timer Enable bit : PWRT disabled
// MCLR Pin Function Select bit : MCLR pin is MCLR function and weak internal pull-up is enabled
// Code Protection bit : Program memory is not code protected
// Brown-out Reset Enable bit : BOR disabled during sleep and enabled during operation
// Flash Program Memory Self Write Enable bits : Write protection off
#pragma config WDTE = ON, PWRTE = OFF, MCLRE = ON, CP = OFF, BOREN = ON, WRT = OFF
#pragma config <setting>=<literal constant>
For example:
// Watchdog Timer Enable bit : WDT enabled
// Power-up Timer Enable bit : PWRT disabled
// MCLR Pin Function Select bit : MCLR pin is MCLR function and weak internal pull-up is enabled
// Code Protection bit : Program memory is not code protected
// Brown-out Reset Enable bit : BOR disabled during sleep and enabled during operation
// Flash Program Memory Self Write Enable bits : Write protection off
#pragma config WDTE = 0x1, PWRTE = 0x1, MCLRE = 0x1, CP = 0x1, BOREN = 0x1, WRT = 0x3
#pragma config <register>=<literal constant>
For example:
// Watchdog Timer Enable bit : WDT enabled
// Power-up Timer Enable bit : PWRT disabled
// MCLR Pin Function Select bit : MCLR pin is MCLR function and weak internal pull-up is enabled
// Code Protection bit : Program memory is not code protected
// Brown-out Reset Enable bit : BOR disabled during sleep and enabled during operation
// Flash Program Memory Self Write Enable bits : Write protection off
#pragma config CONFIG = 0x3FFF
For example:
// IDLOC @ 0x2000
#pragma config IDLOC0 = 0x127

Configuration Registers & Settings

Register: CONFIG @ 0x2007
WDTE Watchdog Timer Enable bit
ON WDT enabled
OFF WDT disabled and can be enabled by SWDTEN bit of the WDTCON register
PWRTE Power-up Timer Enable bit
OFF PWRT disabled
ON PWRT enabled
MCLRE MCLR Pin Function Select bit
ON MCLR pin is MCLR function and weak internal pull-up is enabled
OFF MCLR pin is alternate function, MCLR function is internally disabled
CP Code Protection bit
OFF Program memory is not code protected
ON Program memory is external read and write protected
BOREN Brown-out Reset Enable bit
ON BOR disabled during sleep and enabled during operation
OFF BOR disabled
WRT Flash Program Memory Self Write Enable bits
OFF Write protection off
BOOT 000h to 3FFh write protected, 400h to FFFh may be modified by PMCON1 control
HALF 000h to 7FFh write protected, 800h to FFFh may be modified by PMCON1 control
ALL 000h to FFFh write protected, entire program memory is write protected.
Register: IDLOC0 @ 0x2000
Register: IDLOC1 @ 0x2001
Register: IDLOC2 @ 0x2002
Register: IDLOC3 @ 0x2003
