MPLAB® XC8 C Compiler Toolsuite Master Index

The MPLAB XC8 C Compiler supports the PIC10/12/16/18 microcontrollers (MCUs) and the 8-bit AVR architectures, including tinyAVR and AVR XMEGA. It contains an in-built assembler as well as a stand-alone assembler for PIC devices, packaged as the MPLAB XC8 PIC Assembler.

Data sheets, application notes and other documentation concerning these devices may be found on the Microchip website.

Information on the complier may be found on the MPLAB XC Compiler webpage.

The MPLAB XC8 C Compiler toolsuite consists of:

  • MPLAB® XC8 C Compiler (xc8-cc)
  • MPLAB XC8 Assembler (part of compiler)
  • MPLAB XC8 Object Linker (part of compiler)
  • 8-Bit Utilities (xc8-ar)
  • Hexmate Utility

The MPLAB XC8 PIC Assembler toolsuite is a stand-alone assembler and consists of:

  • MPLAB® XC8 PIC Assembler (pic-as)
  • MPLAB XC8 Object Linker (shared with compiler)
  • 8-Bit Utilities (xc8-ar - shared with compiler)

Documentation for these language tools is described below. Documentation is available in several formats:

  • HTML files may be viewed in your browser window.
  • PDF documents will require the use of the free Adobe Reader to view. PDF documentation may be older than electronic documentation, so view electronic files for the latest information.

Electronic documentation for these language tools may be found on the Online docs website.
